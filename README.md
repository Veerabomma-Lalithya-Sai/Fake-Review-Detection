# Fake Review Detection

## Overview
This project detects fraudulent reviews using Machine Learning and Deep Learning models.

## Models Used
- Random Forest
- Convolutional Neural Network (CNN)

## Features
- Text preprocessing (cleaning, tokenization)
- Feature extraction
- Model training and evaluation

## Tech Stack
- Python
- Scikit-learn
- TensorFlow / Keras
- Pandas, NumPy

## Results
The models were able to classify reviews as real or fake based on textual patterns.
